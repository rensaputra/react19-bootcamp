# react19-bootcamp 🚀

## Introduction

Full-stack code dump from my 2026 React 19 bootcamp. Features Stripe integration, JWT auth, and full CRUD with MySQL. Powered by Next.js & TypeScript.

## Tech used 🏗️

- React
- TypeScript
- NextJS
- Redux and RTK
- Tailwind CSS
- NodeJS
- ExpressJS
- Vite
- SQlite and MySQL
- Prisma ORM
- JOSE + JWT
